Request Data

Individual level sequence data and/or variant calls can be requested as follows.

  1. Fill in the GoNL Data Access Request form. For questions email gonl@bbmri.nl
  2. Your research request will be evaluated by the GoNL steering committee
  3. If positive, you will receive a data access agreementto be signed
  4. You will be granted access to the data using a suitable method
  5. At publication add suitable acknowledgement
