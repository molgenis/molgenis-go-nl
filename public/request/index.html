Request Data

Individual level sequence data and/or variant calls can be requested using the BBMRI's Podium Portal. Visit the Podium Portal to create an account and to learn how to use the system.

The Request Process

  1. Create a Podium account.
  2. Fill in a new data request via Podium. Visit the Podium information page to access the user manual.
  3. Upon submision, your request will be initially checked by the GoNL Coordinator. You will also receive a Podium request number. Take note of this number.
  4. Next, your research request will be evaluated by the GoNL steering committee.
  5. If your request is approved, you will receive a data access agreement to be signed.
  6. You will be granted access to the data using a suitable method.
  7. At publication, add suitable acknowledgement.

You can track the progress of your request in Podium. For questions, email gonl@bbmri.nl.
